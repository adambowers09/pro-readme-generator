# Pro-Readme-Generator

## Description
This progject generates a readme file based on the users input.  Written using JavaScipt and Node.js also features the node package manager inquirer.

## Installation
The server must have Node.js installed.  Run 'npm i' from the root directory to install.

## Usage

Run 'node index.js' from root directory.  Follow the prompts to create content for a readme file.  When completed it will generate a new file, TRIAL_README.md, in the root directory.
